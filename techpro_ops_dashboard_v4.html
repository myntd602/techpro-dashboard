Kết nối Google Sheets
Không cần API Key. Chỉ cần publish Sheet dạng CSV rồi dán link vào đây — xong trong 3 bước.
1
Mở Google Sheet của bạn
Vào Google Sheet đang dùng (hoặc tải file template Excel về → upload lên Google Drive → mở bằng Google Sheets).
Đảm bảo tab đầu tiên có tên KPI_Tháng và cột A = tên chỉ số, cột B = giá trị hiện tại.
2
Publish Sheet dạng CSV (miễn phí, 1 click)
Trong Google Sheet (phiên bản tiếng Việt): File → Chia sẻ → Xuất bản lên web
→ Dropdown trái: chọn tab KPI_Tháng
→ Dropdown phải: chọn "Giá trị được phân cách bằng dấu phẩy (.csv)"
→ Nhấn Xuất bản → Copy link.

✓ Link đúng sẽ có dạng:
.../pub?gid=1690629984&single=true&output=csv

✗ Nếu link chứa /edit hoặc /view là sai — phải lấy từ Xuất bản lên web.
3
Dán link vào đây
Dán đúng link CSV publish từ bước 2. Dashboard sẽ tự đọc data mỗi lần nhấn Sync — không cần làm lại.
Lấy từ File → Chia sẻ → Xuất bản lên web → chọn tab → chọn CSV → copy link
Link đúng sẽ chứa /pub?gid= và output=csv
